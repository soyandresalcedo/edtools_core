# Edtools Core

Custom branding and features for Edtools Educational System.

## Features

- Custom branding (replaces "Frappe" and "ERPNext" with "Edtools")
- Custom login page
- Custom navbar and footer
- Educational workflows optimized for Edtools

## Installation

```bash
bench get-app https://github.com/soyandresalcedo/edtools_core
bench --site [site-name] install-app edtools_core
```
